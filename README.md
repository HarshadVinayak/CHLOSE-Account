# CHLOSE-Account
A Unified Identity and Authentication Infrastructure Layer

## Tech Stack
- React
- TypeScript
- Vite
- Supabase (Backend)
- Framer Motion (Animations)

## Development

This project was bootstrapped with Vite.

### Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
